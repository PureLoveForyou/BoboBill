# 📝 BoboBill - 智能账单管理助手

**移动支付时代，最好的记账方式是不记账！**

Look at you~🤭，是的，你没听错，传统记账软件需要手动输入每笔消费，太麻烦了！在电子支付普及的今天，微信、支付宝、银行APP都已经记录了我们的每一笔交易。BoboBill 的目标不是让你重新记账，而是**智能整合**各个平台的账单数据，让你一目了然地看到自己的消费全貌。

项目正在积极开发中~

## 🎯 项目定位

### 传统记账APP的问题
- 🥺 **无法坚持**：昨天的你信誓旦旦要开始记账，今天的你拍案：谁爱记谁记！
- 🫣 **一分钱也要记？**：是的，机灵的你又想到了，我花2块钱做了地铁，是不是还需要打开手机来记，传统记账就是累泪嘞
- 🤔 **手机不是有自动记账？**：有的有的，VIVO、OPPO等都做了自己记账，忽然有一天，它告诉你：由于不可描述的原因，微信自动记账下架。或者哪天机灵的你想要换手机，哎呀，数据不互通
- 🤡 **APP漏记**：据我所知，没有谁真正做到了全场景自动记账，或多或少都需要手动触发，即使是所谓快捷方式一键记账，还是一样时时刻刻麻烦。你的时间精力值得放在更宝贵的人和事身上！
- 😔 **穿越时空**：本事再大的你，也无法穿越时空。好不容易找到了满意的记账APP，那你这么多年的消费呢，怎么办？不是所有的APP都有账单导入哦

### 为什么需要 BoboBill？
- 📱 **多平台支付**：微信、支付宝、银行卡、信用卡... 账单分散在各个平台
- 📊 **统计困难**：想知道上个月总共花了多少钱？需要打开millions个APP分别查看
- 🕒 **时间浪费**：手动整理账单耗时耗力，很多人干脆不整理了
- 📈 **缺乏洞察**：无法看到消费趋势、分类统计、月度对比

### BoboBill 解决方案
- 🔄 **自动导入**：支持微信、支付宝、银行APP生成的账单文件导入
- 🧩 **智能整合**：自动合并重复交易，统一分类标准
- 📊 **全面统计**：月度/年度统计、分类分析、趋势图表
- 🔍 **高级筛选**：按时间、分类、金额、平台等多维度筛选
- 💡 **消费洞察**：发现消费模式，提供优化建议

## 🚀 快速启动

### 启动后端

```bash
cd backend

# 创建并激活虚拟环境
python -m venv venv
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 启动服务
uvicorn main:app --reload --port 8000

# Windows启动服务
venv\Scripts\uvicorn.exe main:app --reload 
```

### 启动前端

```bash
# 打开另一个终端，回到项目根目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址：http://localhost:5173
```

## 🎨 设计理念

### 与传统记账软件的区别
1. **不重复劳动**：不要求用户手动输入每笔消费
2. **利用现有数据**：微信、支付宝、银行APP已经记录了所有交易
3. **智能整合**：自动合并多平台账单，消除数据孤岛
4. **深度分析**：提供传统记账软件无法实现的跨平台统计

### 用户价值
- ⏱️ **节省时间**：导入账单只需几分钟，整理账单不再痛苦
- 📈 **全面视角**：一眼看清所有平台的消费总和
- 🎯 **消费洞察**：发现消费模式，优化消费习惯
- 🔒 **隐私保护**：数据存储在本地，不上传云端
- 💰 **财务健康**：更好的掌握个人财务状况
